steel::GraphObjectModel

Модель загружается из внешнего файла

class = 'model'

file		path	Path to model in 3DS format
material	material




Tutorial: Как вставить модель в Steel Engine.

1. делаем модель в 3D MAX или дргом редакторе. 
Учтите, что центр модели должен быть в центре координат. 
Обратите внимание на масштаб. Движок все воспринимает в сисеме СИ (1 - значит 1 метр).
2. конвертируем модель в Editable Mesh.
3. если модель состоит из нескольких кусков, то надо из объединить.
4. Используемые материалы не экспортируются, поэтому связь между моделью и движком 
происходит через имя материала. Для простоты назовите ваши материалы model1, model2, ...
где model - имя вашей модели. Имя материала должно состоять не более чем из 8 символов.
5. Экспортируйте только выделенную модель в 3DS файл. В одном 3ds-файле должна быть только одна модель!
6. создайте в scene файле объект:

{
class = 'combiner'
origin = (1 2 3) // смещение

graph = 
{
class = 'model'
file = '<your 3ds file without .3ds extension>'
material = 
{
class = 'std'
diffuse_map =
{ 
	image = '<path to image (bmp, tga)>'
}
}
}
}

7. Каринки только в формате BMP или TGA.
8. Можно не указывать material в конфиге. Тогда движок ищет файл с материалом <model1>.mat 
(model1 - то, как вы назвали материал в редакторе).
9. Запусиите движок. Если ничего не видно, то попробуйте 
a. убрать все остальные объекты
b. вклоючить debug-режим (F3)
c. смотрите steel.log И ищите сообщени об ошибках
