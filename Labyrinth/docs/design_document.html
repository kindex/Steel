Steel Engine

Этот проект использует Steel Engine.

Весь код steel engine, кроме main.cpp и steel.cpp компилируется в библиотеку, которая потом используется в intro. Для Visual C++ 2005 эта библиотека находится в sdk/steel_lib/. Кроме того, в директории steel находятся заголовочные файлы steel engine. demo использует большинство классов и глобальных переменных steel engine.

Для удобства использования в директории steel также находится последняя версия исходников steel engine, и проект steel.vcpoj, который генерирует нужные библиотеки в режиме lib_debug или lib_release.

Intro

По аналогии с классом Steel в Steel Engine, в проекте demo есть класс Demo, который задаёт правила поведения объектов. Этому классу должно хватать игровых объектов, описанных в движке (движки, модели).
