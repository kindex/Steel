Scene (Сцена)

Этот формат файла описывает объекты сцены, их расположение.

Стандартное расширение файла: *.scene

Требования

Формат основан на config.

Частичная совместимость с quake-map форматом.

ООП. В сцене указывается класс. В движке создаётся игровой объект этого класса.

Синтаксис

Вся сцена является struct с классом “scene”.

Все объекты являются struct (в понятии config).
