Steel engine Particle systems 1.0

Этот документ содержит необходимые загружаемые параметры (данные из настроечных файлов) для систем частиц любой сложности и любого типа.

Версия: 0.01 (черновик)

1. Системы частиц без взаимодействия

Short description: Полностью независимые друг от друга частицы. Collision Detection внутри ситемы нет.

2. Системы частиц с заданным взаимодействием

Short description: частицы зависят друг от друга.  Количество частиц  само по себе не меняется. Collision Detection есть. Пример  – твердые тела.  

 

3. Системы частиц с динамическим взаимодействием

Short description: все динамично и меняется с течением времени. Это ХИТРЫЕ системы честиц. Сейчас нам не нужны.

[a15] 

 

Авторы:

            Def. (a@b.c)

 

 [a1]Place, where particles born.

 [a2]В каком направлении излучаются частицы.

 [a3]Как правило это НОРМАЛЬ к плоскости эмиттера. Короче, вектор направленный на «верх».

 [a4]Это означает ФОРМУ эмиттета. Как это задавать - надо еще придумать. Обычно это двух или трехмерные примитивы. Параллелограмм, сфера, эллипсоид.

 [a5]Размер эмиттера.

 [a6]Начальная скорость частицы.

 [a7]Контроль за разбросом параметров.

 [a8]Если true, то умершие частицы возрождаются.

 [a9]Описание материала частицы.

 [a10]Верх частицы. Наверное равен «верху» эмиттера.

 [a11]Размер частицы для рендеринга. Ни на что не влияет.

 [a12]Время жизни честицы после ее рождения.

 [a13]Время для отображеня частицы.

 [a14]Расстояние от эмиттера, на котором частица полностью умирает.

 [a15]Will be described in version 1.2 or later.
