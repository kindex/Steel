Steel Engine Demo [12]

Features:
* custom kinematic
* universal kinematic
* object grouping

Usage:
run bin/steel.exe

Controls:

space	capture/free mousey
Enter	toggle fullscreen/window mode
pause	pause physic model
n	process next frame in paused mode
asdw'/  camera movement
1..9    change physic model speed
F1..F6  toggle draw elements (helper, color, wire, aabb, vertexes, clear)
g	toggle gravity
c	toggle crosshair
mouse1	create sphere
