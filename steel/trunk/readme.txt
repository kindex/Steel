Steel Engine Demo [1.0.13.126]
http://steel.kindex.lv:82/

Features:
* custom kinematic
* universal kinematic
* object grouping

Usage:
bin/steel.exe script=<level.script>
*.bat


Controls:

space	capture/free mousey
Enter	toggle fullscreen/window mode
pause	pause physic model
n	process next frame in paused mode
asdw'/  camera movement
1..9    change physic model speed
F1..F6  toggle draw elements (helper, color, wire, bump, vertexes, clear)
g	toggle gravity
c	toggle crosshair
f	create light
mouse1	create sphere
