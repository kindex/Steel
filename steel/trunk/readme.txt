Steel Engine Demo [11]

Features:
* custom kinematic
* universal kinematic
* object grouping

Usage:
run bin/steel.exe

Controls:

space	capture/free mousey
Enter	toggle fullscreen/window mode
pause	pause physic model
n	process next frame in paused mode
asdw'/  camera movement
1..9    change physic model speed
F1..F5  toggle draw elements (helper, color, wire, aabb, vertexes)
