Форматы файлов

Описывает форматы хранения различных типов ресурсов.

Конфиги

Этот формат файла подходит как для описания сцены, так и для любых конфигов, материалов, анимации, моделей и т.д.

Требования

Текстовый формат.

Вложенность настроек.

Шаблонность. Массив объектов задаётся первым объектом, а в остальных только изменения относительно первого, или вообще любого другого. Таким способом удобно изменять материал, меняя в нём только текстуру.

Без типизации. Объект сам может распознать нужные настройки из строки.

Includes.

Спецификация

В основе конфига лежит понятие value (значение переменной). Всё содержимое файла является value (исключая комментарии).

<value> ::= null

<value> ::= <nuber>

<value> ::= ‘<string>’

<value> ::= “<string>”

<value> ::= {<struct>} | [<tag>] {<struct>}

<value> ::= (<array>)

 

<tag> ::= <id> | <id>:<inherits>

<inherits> ::= <empty> | <external_id>,<inherits>

<id> := virtual <alpha> | <alpha>

<external_id> ::= <alpha> | <path to config>#<alpha>

<struct> ::= <assignment> | <assignment>; <struct> | <empty>

<assignment> ::= <variable> = <value>

<variable> ::= <alpa>

<type> ::= <alpha>

 

<array> ::= <array_element> | <array_element> <array> | <array_element>,<array> |<array_element>; <array> | <empty>

<array_element> ::= <value> | <number>:<value>

 

Комментарии в стиле С++ // и /*.

String – строка, стоящая из любых символов. Символы начала строки (‘ и “) экранируются при помощи \.

Number – целое или дробное число.

alpha - английские буквы и символ подчёркивания. Case sensitive.

Примеры

//some.scene

(

 

     [virtual wall] { // virtual означает, что этот объект служит только как шаблон, и не добавляется в сцену

          material = [:material/diffuse#70] {} // полностью наследуетматериал

          model = ‘model/box’

          scale =(10 10 1)

          texcoord_scale = (0.1 0.1 1)

    }

 

     [w1:wall] { // только дополняем wall

        origin = (10 20 0); // изменяем положение

        material = {

            texture = ‘blick/blick1’ // и текстуру материала

        }

    }

     [w2:wall] {

        origin = (10 20 0);        

        angle = 90;

     }

)

 

// second.scene

(

     [w3:some.scene#w1] {

          origin = (50 50 10);

          model = null; // обнуляет модель предка

          mesh = {

               vertexes = ((0,0,0) (1,0,0) (0,1,0))

               triangles = ((0 1 2)(0 2 1)

          }

     }

)
