О документе

В этом документе описывается порядок и правила работы команды: SVN, встречи, стиль, правила тестирования?

SVN

Исходный код проекта и минимальный набор ресурсов храниться в SVN-репозитории (Subversion). Адрес репозитория: svn+ssh://<user>@kindex.lv:2222/var/svn/steel/steel

Структура репозитория:

·        trunk – текущая стабильно работающая версия проекта.

·        branches – тестовые версии или версии с новыми возможностями, на этапе реализации которых могкт быть неработающие версии.

·        demo – ключевые (констрольные) точки проекта хранятся для истории. Формат директорий: [номер]_[имя].

·        tags – различные интересные версии.

 

При добавлении нового файла не забывайте вставлять шапку, которая описана в соглашении о стилях. Создавайте текстовый файл в кодировке UTF-8.
