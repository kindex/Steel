Канал взаимодействия [kindex]

Канал взаимодействия. Реагируют между собой только те частицы, которые имеют хоть один общий канал. У каждой частицы может быть много каналов. Канал характеризуется типом и силой взаимодействия.

Типы взаимодействия

·        Свободное падение

·        Внешние силы (ветер)

·        Столкновение с полигонами

·        Gravity

·        Spring с рвущимися связями

·        LJ

Кроме того, на каждую частицу действует сила трения.

Модульность [kindex]

Надо иметь возможность заменять следующие модули для каждой системы частиц:

·        Эмиттер (точечный, разброс, равномерный разброс)

·        Renderer (спрайты, система спрайтов, многогранники, меташарики)

·        Physic
