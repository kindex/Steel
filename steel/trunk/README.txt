Controls:

space	capture/free mouse
pause	pause physic model
n	process next frame in paused mode
asdw'/  camera movement
1..9    change physic model speed
F1..F6  toggle draw elements (helper, color, wire, bump, vertexes, clear)
g	toggle gravity
